Race for the Prize

Multi-Video Comparator & Exporter

Upload up to 4 browser recordings (WebM or MOV), synchronize playback, fine-tune start and end points, then export a stitched side-by-side comparison video — all in your browser, no server needed.

Why?

Race for the Prize is a CLI tool that pits browsers against each other in head-to-head performance battles using Playwright. It records video of each browser completing a task, measures execution times, and declares a winner.

But sometimes you already have recordings — from manual testing, different machines, different network conditions, or different tool runs — and you want to compare them visually. This page lets you do that: drop your videos here, align their start and end points, see the stats, and export a single side-by-side video to share with your team.

No uploads leave your machine. Everything runs client-side using FFmpeg WASM for encoding.
